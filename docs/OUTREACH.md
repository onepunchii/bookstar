계산기로 돌아갑니다. 문제는 이 방식이 사람의 꼼꼼함에 기대기 때문에,
단 한 번의 누락이 방송 펑크가 된다는 겁니다.

xong는 소속사를 위한 부킹 OS입니다.

- 데일리 스케줄표를 콜타임·장소·차량까지 원클릭 카톡 전파
- 가용성 캘린더·홀드가 더블부킹을 자동 차단
- 분배율·원천징수 3.3% 자동 정산 + 미수금 리마인더
- 섭외 성사 수수료 0% — 저희는 브로커가 아닙니다

5팀까지는 영구 무료라 도입 리스크가 없습니다. 3분이면 어떤 물건인지

연예인 섭외 견적에는 평균 38%의 다단계 마진이 붙습니다.
