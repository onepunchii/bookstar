# 기획사 아티스트 관리 실태 리서치 (2026-07)

xong 소속사 센터가 능가해야 할 대상 조사 요약.

## 핵심 발견

1. **한국 기획사의 표준 도구는 "카카오톡 + 엑셀/수기 + 공유 캘린더"다.**
   전문 SaaS가 없다. 대형사는 자체 그룹웨어를 쓰지만 시장 전체를 커버하는 표준 도구는 부재.
   2019년 '호두'라는 일정 공유 앱이 이 공백을 노렸으나 일정 공유 단일 기능에 그침.
   매니저의 최대 공포는 "스케줄 누락 = 방송/공연 펑크".

2. **섭외 시장에는 평균 38%의 다단계 브로커 거품이 있다.**
   광고대행사(+15%) → 행사대행사(+20%) → 직거래 에이전시(+10%) 구조.
   실출연료 3,500만원짜리가 5,313만원까지 부풀어짐 (대학축제 기준).
   → 소속사 직영 공식 창구(=xong)가 구조적으로 가치 있음.

3. **국내 경쟁 서비스는 전부 "수요자 중심 마켓플레이스"거나 "브로커"다.**
   - 쇼글: 최대 섭외 플랫폼(공연팀 2,772팀), 공고→지원·견적 비교. 소속사용 OS 아님
   - 헤이비글: 역경매 견적 비교, 웨딩·소규모 행사 중심
   - 콜마이스타/아츠로/탑스매치: 사람이 중개하는 전통 대행사 (38% 거품의 당사자)
   - 레뷰/마켓잇: 인플루언서 캠페인 특화, 연예인급 섭외·계약·정산 없음
   - 필름메이커스/플필: 배우 캐스팅 공고 게시판
   → **"부킹 에이전시 OS"(인입→견적→계약→일정→정산 일체형)는 국내 공백 시장**

4. **해외 검증된 뼈대** (Gigwell, Prism.fm, SystemOne, Overture, Master Tour, Stagent):
   인입(부킹 위젯/EPK) → 홀드/오퍼 → 계약(e-sign) → 어드밴싱 → 일정/이동 → 인보이스 → 정산/커미션.
   차별화 포인트: 감사추적(Prism), 자동 정산(Prism), CRM 자동화(Overture),
   AI 문서 추출(Overture AI: PDF/엑셀 → 날짜·출연료·연락처 자동 추출).

## 능가하기 위한 기능 우선순위

### P0 — 요청이 돈이 되기까지의 병목 제거
1. **섭외 인입 인박스 통합** — 부킹 페이지 + 이메일 + 카톡채널 문의를 하나의 파이프라인으로
2. **홀드(가일정) 시스템** — 가확정/1차 홀드/확정 + 자동 만료 알림 + 더블부킹 실시간 차단 (Prism의 킬러 기능, 카톡+엑셀로 불가능)
3. **견적서 원클릭 생성** — 아티스트별 출연료·조건 프리셋 → 견적 PDF/링크 자동 생성 + 열람 추적
4. **직거래 신뢰 레이어** — 주최자 검증, 견적 감사추적, 표준계약+전자서명, 에스크로

### P1 — 반복 업무 자동화 (리텐션)
5. **카톡-네이티브 일정 전파** — 확정 일정을 알림톡으로 자동 전파(콜타임·장소·내비링크) + 읽음 확인
6. **AI 문서 인테이크** — 공문/큐시트 PDF·엑셀 → 요청 카드 자동 생성
7. **정산·커미션 자동화** — 소속사:아티스트 분배율, 원천징수 3.3%, 세금계산서, 미수금 리마인더

### P2 — 데이터 해자
8. **가용성 캘린더 외부 공유 링크** — 검증된 주최자에게 가능/불가만 공개
9. **어드밴싱 체크리스트** — 행사 전 준비사항(라이더·대기실·주차) 공유 링크
10. **로스터 성과 대시보드** — 문의→성사 전환율, 단가 추이, 수요 히트맵 → 프라이싱 인사이트
